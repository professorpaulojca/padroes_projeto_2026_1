UX

Mapa de Fluxos

Caminhos principais de navegação e relações entre entidades.

🔐 Fluxo de Autenticação

01 Tela de Login
├ Esqueci minha senha → Redefinir
└ Novo aqui? → Cadastro
02 Dashboard Principal
03 Perfil → Associar Pessoa

👥 Fluxo de Gestão

01 Menu → Pessoas
├ Lista → Criar Pessoa
└ Lista → Editar Pessoa (+ extensões)
02 Pessoa → Endereços
├ Adicionar endereço (ViaCEP)
└ Múltiplas pessoas por endereço

📐 Regras de Negócio (relações)

Usuário ↔ Pessoa

Um usuário pode ou não estar associado a uma Pessoa.
Uma Pessoa pode ou não ter usuário.

Pessoa ↔ Endereço

Uma Pessoa pode ter vários endereços.
Um Endereço pode pertencer a uma ou mais Pessoas.

Pessoa → Extensões

Tipo sanguíneo, veículos, pets e outros
dados são módulos opcionais por pessoa.
